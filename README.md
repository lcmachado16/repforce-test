## Tarefa 1 — Página de Catálogo (Listings)

```
Tarefa 1 — Página de Catálogo (Listings)
Crie a tela inicial com filtros funcionais e a listagem de produtos.
Filtros
A barra de filtros deve ter:
• Categoria (select com as categorias disponíveis nos dados)
• Marca (select)
• Faixa de preço (slider duplo ou dois inputs min / max)
• Apenas em estoque (checkbox — stock > 0)
• Busca por texto (input que filtra por title)
• Botão Limpar filtros
Os filtros devem funcionar em conjunto (combinados) e atualizar a listagem em tempo real. O
estado dos filtros deve estar refletido na URL (query params) para que a navegação seja restaurável.
```
